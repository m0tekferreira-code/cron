A Grande Analogia 🤖

Bem-vindo ao treinamento! Para começar, vamos desmistificar o conceito. Imagine que o seu servidor é uma grande empresa e o CRON é o funcionário mais disciplinado (e obsessivo) do mundo.

O Perfil do Funcionário CRON:

  • Não dorme.
  • Não toma café.
  • Olha para o relógio a cada 60 segundos.
  • Executa ordens exatas no segundo exato.

"Quando dá a hora exata que você combinou, ele aperta o botão e volta a olhar para o relógio."

Laboratório de Sintaxe 🧪

Um Cronjob é formado por 5 asteriscos. A melhor forma de decorar é ler da menor unidade para a maior (esquerda para direita).

Minuto (0-59)
Hora (0-23)
Dia Mês (1-31)
Mês (1-12)
Semana (0-7)

Tradução Humana

Executar a cada minuto.

Dicas de Sênior 🧙‍♂️

Saber a sintaxe é fácil. O difícil é não quebrar a produção. Abaixo, uma visualização real do "Perigo da Meia-Noite".

Erro Comum: 0 0 * * *

Todo mundo quer rodar backups à meia-noite. Se você tiver 50 tarefas rodando ao mesmo tempo, o servidor trava (CPU Spike).

Solução: Distribuição

Espalhe as tarefas! 00:05, 02:30, 04:15. O servidor agradece.

Caminhos Absolutos

Nunca use ./script.sh. O Cron não sabe onde ele está. Use /home/user/script.sh.
